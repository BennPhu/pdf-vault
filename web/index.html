  • Add files — drop PDFs or images (PNG/JPEG/WebP) onto the drop zone, or click to browse. Images are converted to PDFs automatically.
    • 
          
  • Search — type in the box atop the Library; press Esc to clear.
    • 
          
  • Rename — double-click a file's name in the Library.
    • 
          
  • Preview — click any PDF to flip through its pages on the right.
    • 
          
  • Merge / Split — select files, then use the bottom bar: merge into one, save a page range, or split every page to its own file.
    • 
          
  • Edit Pages — select one PDF to rotate, reorder, or delete pages.
    • 
          
  • Compress — shrink selected PDFs in place; originals are kept in trash so you can undo.
    • 
          
  • Delete & Undo — deleted files go to a hidden trash for 30 days; use Undo/Redo in the Library header.
    • 
          
  • Master PDF — bind your entire library into a single document.
    • 
          
  • Storage — everything lives in the folder you chose; change it anytime via ⚙ Storage.
    • 
        

      

      

        
PDF Vault exists so you never have to upload a document to "just merge two PDFs" again.

        
Our mission is a light, fast PDF library that is 100% local: your files never leave your machine, there are no accounts, no ads, no telemetry, and no cloud. The app is free and open source, so anyone can verify exactly what it does.

        
The only network request the app ever makes is an optional, checksum-verified update check against GitHub Releases.
